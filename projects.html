Projects

A collection of open source projects I've been working on.

gh-issue-types

A GitHub CLI extension for managing issue types. Query, view, and set issue types on GitHub issues directly from your terminal. Supports listing available types, getting/setting types on issues, filtering issues by type, and enhanced issue views with type information.

Coding Agent Progress Manager

A CLI tool for tracking lightweight requirements across runs of a coding agent. Inspired by Anthropic's blog post on effective harnesses for long-running agents, it provides a simple JSON-based progress tracking system with commands to initialize, add, update, and manage requirements. Includes a web UI and specialized prompt templates for different analysis types.
